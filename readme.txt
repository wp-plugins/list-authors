A widget to display a list of site authors.

== Description ==

A sidebar widget to display a list of authors in your WordPress blog. Based off of [List Authors Widget 1.0](http://www.mikesmullin.com/2006/06/21/list-authors-widget-10) by Mike Smullin. Improvements include a revised options screen (uses checkbox input fields instead of free text where appropriate) and the ability to include links to authors' RSS feeds.

#### Features:
* Can show number of published author posts.
* Allows administrator to be excluded from the list.
* Choose between displaying usernames or full names in the list.
* Allows users with 0 posts to be excluded from the list.
* Can include links to author-specific RSS feeds.


== Installation ==

1. Download the List Authors zip file.
2. Extract the files to your WordPress plugins directory.
3. Activate the plugin via the WordPress Plugins tab.
4. Configure the widget as you wish and place it somewhere in your sidebar.
